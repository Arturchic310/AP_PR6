Корткий опис проєкту PR6:
--------------------------------------------------------------------------------------------------------
Цей проєкт демонструє використання асинхронності в Spring з використанням анотації `@Scheduled` та класу
`ScheduledExecutorService`. Проєкт створює файл `counter.txt` і періодично оновлює його значення, 
виконуючи асинхронні задачі.
--------------------------------------------------------------------------------------------------------
Основні класи та методи
 - `AsyncSpringApplication` — головний клас, що ініціалізує застосунок та асинхронні задачі.
 - Метод `run` використовує `ScheduledExecutorService` для запуску задач кожні 10 секунд.
 - `FileUpdater` — компонент, що оновлює файл.
 - Метод `initFile` — створює або очищує файл при запуску програми.
 - Метод `updateFile` — записує значення лічильника у файл.
--------------------------------------------------------------------------------------------------------
Основний функціонал:
 1. Файл `counter.txt` створюється, якщо його немає, і очищується при запуску програми.
 2. Кожні 10 секунд значення лічильника збільшується на 5, а результат записується у файл.
 3. Через 5 секунд після запуску програми в консоль виводиться повідомлення.
--------------------------------------------------------------------------------------------------------
Можливі покращення
 - Зробити шлях до файлу конфігурованим через `application.properties`.
 - Реалізувати обробку виключень більш детально.
 - Додати тести для основної функціональності.
